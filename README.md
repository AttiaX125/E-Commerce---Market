# 🛒 Ecommerce Web App

A modern full-stack ecommerce platform built using **Next.js, React, and REST APIs**, designed with a premium **frosted glass + sand-themed UI**.

The project includes full shopping flow: products, cart, checkout, orders, and reviews system.

---

## ✨ Features

- 🛍️ Product listing with dynamic API data
- 🔎 Product details page
- 🛒 Cart system (add / update / remove items)
- 📦 Order history page
- ⭐ Reviews system (create, update, delete)
- 👤 User authentication (login / register)
- 🔐 Protected routes
- 📱 Fully responsive design (mobile-first)
- 🎨 Premium UI (frosted glass + sand aesthetic)
- ⚡ Fast performance with Next.js optimization

---

## 🧱 Tech Stack

- **Frontend:** Next.js, React, TypeScript
- **Styling:** Tailwind CSS (or your styling system)
- **State Management:** React hooks / Context API
- **API:** RESTful backend integration
- **Authentication:** JWT-based auth
- **Deployment:** (Vercel / GitHub)
